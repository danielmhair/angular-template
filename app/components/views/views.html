My Views

This is the 'views' component. It's state is 'root.views'.
For us to show you this component (the 'views' component), we added an attribute in our root-component called 'ui-view' to a 'div' tag (you could add 'ui-view' to any tag you want; we chose a 'div'). This tells angular that any children of that state will be put inside that 'div'. Since this state ('root.views') has the parent state in its name ('root'), then it is a child of the 'root' state.
